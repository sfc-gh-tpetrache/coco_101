# Ticket Classification Taxonomy

Defines the allowed labels for classification using AI_CLASSIFY.

## product_category
- Billing
- Authentication
- Performance
- UI/UX
- Integrations
- Data Platform
- Security
- Other

## issue_type
- Bug
- Feature Request
- Configuration Issue
- Data Issue
- Usage Question
- Access Problem
- Unknown

## priority_bucket
- P0
- P1
- P2
- P3

## Rules
- These labels are used ONLY with AI_CLASSIFY
- Labels must remain stable for aggregation and dashboards
- Do not generate these fields with AI_COMPLETE
- If uncertain, use "Other" or "Unknown"
